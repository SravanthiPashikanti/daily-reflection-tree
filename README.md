# Daily Reflection Tree

## 📌 Problem Statement

This project helps users reflect on their daily challenges using a structured decision tree and provides actionable suggestions.

---

## 🌳 How It Works

The system asks a series of questions based on the user's emotional or productivity state and guides them to a final action and reflection.

---

## 🌿 Decision Tree Overview

The tree starts with:
**"How are you feeling today?"**

Based on the response, it branches into:

* Stressed
* Unproductive
* Distracted

Each branch leads to specific actions and reflection questions.

---

## 🧪 Example

**Input Path:**
Stressed → Work → Too many tasks

**Output:**
Action: List top 3 priorities and focus on one
Reflection: What truly matters today?

---

## 🛡️ Guardrails

* Only predefined options are allowed (no random AI answers)
* No medical or psychological advice is given
* Each path ends with a clear action and reflection
* The system avoids ambiguity by guiding users step-by-step
